## Patterns
- Cluster-aware host helpers (`isAppsClusterHost`, `isLegacyClusterHost`, `authHubOrigin`, `appsCatalogOrigin`) centralize multi-cluster routing logic so proxy/auth/redirect code can branch on cluster without re-parsing hosts
- Subdomain-shape invariant test (`every subdomain is a single DNS label (no dots)`) prevents future drift where someone might store a full host in `app.subdomain` and break the leftmost-label extraction
