## Patterns
- Migrating ad-hoc `<input>`/`<label>`/`<button>` to `@repo/ui` primitives while preserving existing Tailwind classes via `className` prop keeps visual parity with zero regression risk.
- Test-only `vi.mock("next/link", ...)` with an `<a>` forward lets component tests assert `href` without NextRouter context.
- Adding a marker-delimited canonical listing (`<!-- lib-listing:start -->`) in CLAUDE.md plus a CI script is a lightweight way to enforce doc/code sync.

## Anti-Patterns
- Deleting a review findings file without leaving a deferred-work note makes it easy to lose track of info-severity items — prefer explicit "unfixed/deferred" entries.
- Wrapping `@repo/ui` primitives in thin local shims (e.g. local `Label`, `Input`) that just re-forward props adds indirection without value; consume the primitive directly when possible.
