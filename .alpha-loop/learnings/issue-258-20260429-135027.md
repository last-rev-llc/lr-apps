- Cluster-aware host helpers (`appHost`, `appOrigin`, `authHubOrigin`, `appsCatalogOrigin`, `isAppsClusterHost`, `isLegacyClusterHost`) parameterized by current host enable parallel prod/legacy/local cluster support without branching at every call site
