# Superstars

**URL:** https://superstars.adam-harris.alphaclaw.app

Personal tribute pages with rich animations. Multi-person ready (hash-based navigation).

## Data
- `data/people.json` — Sam Reynolds and Carla Reynolds
- Education field MUST be array of objects `[{school, degree, sport, years}]`, not a string (caused render bug)

## Features
Confetti, particles, floating emoji sprites, animated stat counters, scroll-triggered fade-ins, photo gallery with lightbox, embedded YouTube videos, career timeline, team cards, articles, quotes.
