Data Format

Each saved item in the cringe_rizzler Supabase table has the following fields:

Field Type Description
idTextUnique ID (phrase-timestamp or meme-timestamp)
typeText"phrase" or "meme"
contentTextThe phrase text or meme caption
slang_termsJSONBArray of slang terms used
scenarioTextMeme scenario (memes only)
image_urlTextCanvas-rendered meme image as data URL (memes only)
created_atTimestampWhen the item was saved

API Integration

The app uses the following APIs:

Phrase Generation: POST /v1/chat/completions (gpt-4o-mini) Meme Captions: POST /v1/chat/completions (gpt-4o-mini) Meme Templates: GET https://api.imgflip.com/get_memes (top 100 templates)

OpenAI API key is read from a <meta name="openai-key"> tag in the page head. Meme images use classic imgflip templates with AI-generated caption text overlaid via canvas.
