💜 Sentiment Documentation

Track team mood, blockers, and highlights with real-time sentiment analysis.

## Getting Started Sentiment tracks daily team health through mood scores, blockers, and highlights. ### Quick Start 1. Open the [app](app.html) 2. View team sentiment dashboard 3. Filter by member or view full timeline 4. Monitor trends and identify blockers early ### Key Features - **Real-time sentiment tracking** — live updates from Supabase - **Trend visualization** — interactive Chart.js graphs - **Blocker detection** — automatic flagging of blocked team members - **Highlight tracking** — celebrate wins and positive momentum - **Member filtering** — focus on individual or team-wide views
## Components ### `` Main dashboard component displaying sentiment data, trends, and member cards. **Location:** `modules/sentiment.js` **Features:** - Stats overview (avg sentiment, total entries, blocked days, highlights) - Interactive trend chart with Chart.js - Member cards with latest mood and sentiment bar - Daily timeline with all entries - Member filter dropdown **Data Source:** Supabase `sentiment_entries` table **Usage:** ```html ``` ### `` Dashboard widget showing team sentiment summary. **Location:** `modules/sentiment-summary.js` **Usage:** ```html ```
## Supabase Schema ### `sentiment_entries` Table | Column | Type | Description | |--------|------|-------------| | `id` | text | Unique entry ID (e.g., s001) | | `date` | date | Entry date | | `member_name` | text | Team member name | | `sentiment_score` | int4 | Score 0-10 | | `mood` | text | Mood label (positive, neutral, frustrated, blocked, excited) | | `work_summary` | text | Brief work summary | | `blockers` | jsonb | Array of blocker strings | | `highlights` | jsonb | Array of highlight strings | | `created_at` | timestamptz | Auto-generated timestamp | **RLS Policy:** Authenticated users can read all entries. **Query Example:** ```javascript const { data, error } = await supabase .from('sentiment_entries') .select('*') .order('date', { ascending: false }); ```

## Troubleshooting ### Data not loading - Check Supabase connection in browser console - Verify `sentiment_entries` table exists - Confirm RLS policies allow authenticated reads ### Authentication issues - Clear browser cache and cookies - Check Supabase auth configuration - Verify publishable key in meta tags ### Chart not rendering - Confirm Chart.js loaded (check Network tab) - Check browser console for errors - Verify data format matches Chart.js expectations
