📋 Queue

Trigger queue viewer with multiple view modes, status filters, infinite scroll, and item detail modals with revert capability

Usage

<cc-queue></cc-queue>
AttributeTypeDescription
No attributes — loads from Supabase trigger_queue table automatically

Features

  • 3 view modes: Cards (default), Vertical Timeline, Horizontal Timeline (swim lanes)
  • Status filters: Active, Pending, Processing, Done, Failed, All
  • Infinite scroll: Loads 50 items at a time with IntersectionObserver sentinel
  • Auto-refresh: Polls every 8 seconds when tab is focused
  • Detail modal: Click any item for full message, timestamps, and status
  • Revert: Done items can be reverted — creates a new pending job with undo instructions
  • Copy ID: Click the short ID chip to copy the full UUID

📦 Batches

Batch group management with search, view toggle, detail modals, and batch execution

Usage

<cc-batches></cc-batches>
AttributeTypeDescription
No attributes — loads from Supabase cron_batches, cron_jobs, and cron_batch_jobs tables automatically

Features

  • 3 view modes: Cards (default), List (compact), Expanded (full detail with job pills)
  • Search: Filter by batch name or schedule
  • Detail modal: Click a batch to see schedule info, assigned jobs, and enable/disable or run
  • Run batch: Queues all enabled jobs in the batch via trigger queue
  • Enable/disable: Toggle batch enabled state inline or from detail modal
