Data Sources

Gen Alpha: Loaded from the slang Supabase table (same as the Gen Alpha app).
Gen X: Loaded from data/gen-x-slang.json (static JSON file).

Both sources are merged with a generation field (gen-alpha or gen-x) and can be filtered, searched, and compared.

SlangTranslatorDB API

const db = await SlangTranslatorDB.init();

// Get all slang (merged)
const all = await db.getAll();

// Filter by generation
const alphaOnly = await db.getAll({ generation: 'gen-alpha' });

// Search
const results = await db.getAll({ search: 'rizz' });

// Get categories
const cats = await db.getCategories();

// Get cross-generation pairs map
const pairs = db.getPairs();

Features

  • Dictionary: Unified card grid with generation/category filters and search
  • Translator: Two-panel live translation between Gen Alpha and Gen X
  • Compare: Side-by-side matched pairs across generations
  • Quiz: Cross-generational multiple choice with score tracking
