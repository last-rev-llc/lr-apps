📊 Pour Summary

Compact dashboard widget showing restaurant pour statistics at a glance.

Usage

<proper-wine-pour-summary src="data/restaurants.json"></proper-wine-pour-summary>

Attributes

AttributeTypeDefaultDescription
srcstring—URL to restaurant JSON data file

What It Shows

  • Count of generous restaurants
  • Count of stingy restaurants
  • Total tracked restaurants
  • Average glass price

Live Example

Embed in Your Command Center

<!-- 1. Load shared theme + components --> <link rel="stylesheet" href="https://shared.adam-harris.alphaclaw.app/theme.css"> <script src="https://shared.adam-harris.alphaclaw.app/components/index.js"></script> <!-- 2. Load the module --> <script src="https://proper-wine-pour.adam-harris.alphaclaw.app/modules/proper-wine-pour-summary.js"></script> <!-- 3. Use it --> <proper-wine-pour-summary src="https://proper-wine-pour.adam-harris.alphaclaw.app/data/restaurants.json"></proper-wine-pour-summary>

📄 Data Format

The JSON structure expected by modules.

Schema

FieldTypeDescription
idstringUnique slug identifier
namestringRestaurant name
neighborhoodstringSF neighborhood
wine_list_ratingnumber1-5 rating of wine list quality
pour_ratingstring"generous" | "standard" | "stingy" | "criminal"
avg_glass_pricenumberAverage price per glass ($)
notesstringComments about the restaurant
visitsarrayArray of visit objects with date, wine, rating, price
Example JSON 
[
  {
    "id": "gary-danko",
    "name": "Gary Danko",
    "neighborhood": "Russian Hill",
    "wine_list_rating": 5,
    "pour_rating": "generous",
    "avg_glass_price": 22,
    "notes": "Consistently excellent pours.",
    "visits": [
      { "date": "2024-11-15", "wine": "Opus One", "rating": 5, "price": 45 }
    ]
  }
]
