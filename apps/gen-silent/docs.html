Embeddable modules for your command center dashboard.

Summary Widget

A compact dashboard widget showing term count, average vibe score, top terms, and category breakdown.

Usage

<gen-silent-summary src="data/slang.json"></gen-silent-summary>

Attributes

Attribute Type Description
src String Path to slang JSON data file

Live Example

Embed in Your Command Center

<script src="https://gen-silent.adam-harris.alphaclaw.app/modules/gen-silent-summary.js"></script> <gen-silent-summary src="https://gen-silent.adam-harris.alphaclaw.app/data/slang.json"></gen-silent-summary>

Data Format

Each slang term in data/slang.json has the following fields:

Field Type Description
idStringUnique slug identifier
termStringDisplay name of the slang term
definitionStringWhat the term means
exampleStringExample sentence using the term
categoryStringOne of: Approval, Disapproval, Greeting, Lifestyle, Insult, Reaction, Romance
vibeScoreNumber1-10 rating of how hard the term hits
originStringWhere the term originated
aliasesArrayAlternative spellings or variations
eraStringTime period (1940s, 1950s)
equivalentsObjectCross-generational equivalent terms

Categories

Approval Insult Reaction Lifestyle Disapproval Greeting Romance

Theming

The component uses standard theme.css variables with a dark goldenrod (#b8860b) accent. Category colors:

Approval: #22c55e Insult: #ef4444 Reaction: #eab308 Lifestyle: #8b5cf6 Disapproval: #f97316 Greeting: #06b6d4 Romance: #ec4899

Vibe score bars use a 10-color gradient scale from red (1) to green (10).
