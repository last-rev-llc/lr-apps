Under the hood

Not a Chatbot. An Operator.

AlphaClaw doesn't just answer questions — it runs shell commands, manages git repos, deploys code, queries databases, browses the web, and orchestrates multi-step workflows with sub-agents.

Full Shell Access

Runs npm, git, curl, jq, ffmpeg — any CLI tool. Installs packages, writes scripts, manages processes. Not sandboxed to a chat window.

Sub-Agent Orchestration

Spawns background agents for heavy tasks — deep code reviews, batch research, multi-repo audits — while staying responsive to you in chat.

Persistent Workspace

A real filesystem with git-tracked workspace. Apps, skills, memory files, scripts — everything persists across sessions and survives restarts.

Browser Automation

Controls a real browser — scrapes pages, fills forms, captures screenshots, navigates SPAs. Research and automation beyond what APIs expose.
