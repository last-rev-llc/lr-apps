📊 Concert Summary

Compact dashboard widget showing concert counts and upcoming shows.

Usage

<cc-concerts-summary></cc-concerts-summary>

Attributes

AttributeTypeDefaultDescription
———No attributes required. Loads data from Supabase automatically.

What It Shows

  • Total concert count
  • Number of shows you're going to
  • Upcoming shows count
  • Next 5 upcoming concerts with artist, venue, and date

Live Example

Embed in Your Command Center

<!-- 1. 📄 Data Format

Supabase table schemas used by the concert modules.

concerts Table

FieldTypeDescription
idtextPrimary key
artisttextArtist/band name
venuetextVenue name
citytextCity
datetextISO date (YYYY-MM-DD)
timetextShow time (HH:MM)
ticket_urltextLink to buy tickets
genretextGenre tag
notestextAdditional notes
created_bytextWho added the concert

concert_rsvps Table

FieldTypeDescription
idtextPrimary key
concert_idtextFK to concerts.id
user_nametextUser's display name
user_initialstext2-letter initials for avatar
statustextgoing / maybe / not-going
