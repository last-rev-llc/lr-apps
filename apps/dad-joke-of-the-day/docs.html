🤣 Dad Joke of the Day — Module Docs

Embeddable modules you can add to your command center or any page.

📊 Summary Widget

A compact card that shows today's joke of the day with a click-to-reveal punchline. Perfect for dashboards.

Usage

<dad-joke-of-the-day-summary src="data/jokes.json"></dad-joke-of-the-day-summary>

Attributes

AttributeTypeDefaultDescription
srcstring—Path to the jokes JSON file

What It Shows

  • Today's joke setup text
  • Click-to-reveal punchline
  • Joke category badge
  • Rating buttons (😩 Groan-worthy, 🙄 Eye-roll, 😂 Actually funny)

Live Example

Embed in Your Command Center

<!-- 1. Load shared theme + components --> <link rel="stylesheet" href="/apps/shared/theme.css"> <script src="/apps/shared/components/index.js"></script> <!-- 2. Load the module --> <script src="https://dad-joke-of-the-day.adam-harris.alphaclaw.app/modules/dad-joke-of-the-day-summary.js"></script> <!-- 3. Use it --> <dad-joke-of-the-day-summary src="https://dad-joke-of-the-day.adam-harris.alphaclaw.app/data/jokes.json"></dad-joke-of-the-day-summary>

📄 Data Format

The JSON structure expected by modules. Stored in data/jokes.json.

Schema

FieldTypeDescription
idnumberUnique joke ID
setupstringThe joke question/setup
punchlinestringThe punchline/answer
categorystringOne of: Classic, Food, Animals, Tech, Holiday, Work, Science
Example JSON 
[
  {
    "id": 1,
    "setup": "Why don't scientists trust atoms?",
    "punchline": "Because they make up everything!",
    "category": "Science"
  },
  {
    "id": 2,
    "setup": "What do you call a fake noodle?",
    "punchline": "An impasta!",
    "category": "Food"
  }
]
