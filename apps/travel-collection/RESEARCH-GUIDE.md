# Travel Collection — Property Research Guidelines

## Photo Sourcing Rules

**ALWAYS source photos from the hotel's official website.**

1. Go to the property's official website first
2. Pull images directly from their site (media/gallery pages, homepage hero images)
3. Acceptable secondary sources: the hotel's official social media, press kits, or verified booking sites (e.g., Mr & Mrs Smith, Virtuoso) that use the hotel's own photography
4. **NEVER use**: stock photos (Unsplash, Pexels, Shutterstock), generic destination images, AI-generated images, or images from unrelated properties
5. Verify the image actually shows the specific property — not a different property in the same chain or location
6. Prefer high-resolution images (1200px+ wide) with good composition

## Verification Checklist
- [ ] Image URL domain matches or is clearly affiliated with the property
- [ ] Image shows the actual property (not a generic beach/mountain/city shot)
- [ ] If using a booking site photo, confirm it's tagged to the correct property
- [ ] For chains with multiple locations, confirm the image is from the correct location
