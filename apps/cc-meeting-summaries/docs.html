📝 Meeting Summaries — Documentation

Overview

Meeting Summaries automatically processes Zoom cloud recordings into structured intelligence: summaries, decisions, action items, and sentiment analysis.

Data Pipeline

  1. Zoom Recording — Meetings recorded via Zoom cloud
  2. Transcript Download — VTT transcripts fetched via Zoom API (Server-to-Server OAuth)
  3. AI Processing — Claude analyzes transcripts for summaries, decisions, action items
  4. Storage — Results stored in Supabase zoom_transcripts table
  5. Display — This app renders the data with search, filters, and action buttons

Processing Meetings

To process new meetings, use the zoom-meeting-intelligence skill via Slack:

/summarize yesterday's meetings
/summarize the Diligent call from last week

Action Items

Action items extracted from meetings include:

  • Owner — Person responsible
  • Priority — High / Medium / Low
  • Deadline — When it's due
  • Create Ticket — Generates a JIRA ticket command
  • Follow-up — Generates a follow-up email prompt
