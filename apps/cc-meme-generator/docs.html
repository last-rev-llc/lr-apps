cc-meme-generator

The main web component that powers the meme generator. Fetches templates, renders text on canvas, and exports PNGs.

Usage

<cc-meme-generator></cc-meme-generator>

Features

  • Loads 100 popular meme templates from Imgflip API
  • Search/filter templates by name
  • HTML5 Canvas-based text rendering
  • Impact font with white fill and black stroke
  • Adjustable font size (16–80px)
  • Auto text wrapping for long captions
  • One-click PNG download

Imgflip API

Templates are fetched from the free Imgflip API. No authentication required for the template list endpoint.

Endpoint

GET https://api.imgflip.com/get_memes Response: { success: true, data: { memes: [...] } } Each meme object: - id: string - name: string (e.g. "Drake Hotline Bling") - url: string (image URL) - width: number - height: number - box_count: number (number of text boxes)

Canvas Rendering

Text is rendered directly onto an HTML5 Canvas element overlaid on the template image.

CORS Notes

Imgflip images are loaded with crossOrigin="anonymous". If CORS headers are present, the canvas can be exported as PNG. If CORS is blocked, the image loads as a fallback but canvas export (toDataURL) may throw a security error. In that case, users can right-click the canvas to save.
