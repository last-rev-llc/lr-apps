Embeddable modules for your command center dashboard.

📊 Summary Widget

A compact dashboard widget showing term count, average vibe score, top terms, and category breakdown.

Usage

<gen-boomers-summary src="data/slang.json"></gen-boomers-summary>

Attributes

Attribute Type Description
src String Path to slang JSON data file

Live Example

Embed in Your Command Center

<script src="https://gen-boomers.adam-harris.alphaclaw.app/modules/gen-boomers-summary.js"></script> <gen-boomers-summary src="https://gen-boomers.adam-harris.alphaclaw.app/data/slang.json"></gen-boomers-summary>

📄 Data Format

Each slang term in data/slang.json has the following fields:

Field Type Description
idStringUnique slug identifier
termStringDisplay name of the slang term
definitionStringWhat the term means
exampleStringExample sentence using the term
categoryStringOne of: Approval, Disapproval, Greeting, Lifestyle, Compliment, Reaction
vibeScoreNumber1-10 rating of how groovy the term is
originStringWhere the term originated
aliasesArrayAlternative spellings or variations
eraStringTime period the term emerged (1960s-1970s)
equivalentsObjectCross-generation equivalents (genAlpha, genX)

🏷️ Categories

✌️ Approval 👎 Disapproval ⚡ Reaction 🌸 Lifestyle 💕 Compliment 🤝 Greeting

🎨 Theming

The component uses standard theme.css variables. Theme color: #ff6347 (tomato). Category colors:

Approval: #22c55e Disapproval: #ef4444 Reaction: #eab308 Lifestyle: #ff6347 Compliment: #ec4899 Greeting: #06b6d4

Vibe score bars use a 10-color gradient scale from red (1) to green (10).
