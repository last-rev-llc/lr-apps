📊 Ideas Summary

A compact dashboard widget that shows your top-scored ideas at a glance. Designed for overview pages, daily standup screens, or any dashboard that needs an ideas pulse.

Usage

<cc-ideas-summary src="data/ideas-summary.json"></cc-ideas-summary>

Attributes

AttributeTypeDefaultDescription
srcstring—URL to JSON data file (same format as ideas.json)

What It Shows

  • Top ideas filtered by: your 5-star ratings OR composite score ≥ 9
  • Category color pills (Product, Content, Business, Technical, Creative)
  • Composite score badge per idea
  • Total active idea count
  • "View all →" link to the full board

Live Example

Embed in Your Command Center

<!-- 1. Include shared components (if not already loaded) --> <link rel="stylesheet" href="https://shared.adam-harris.alphaclaw.app/theme.css"> <script src="https://shared.adam-harris.alphaclaw.app/components/index.js"></script> <!-- 2. 📄 Data Format

The JSON structure expected by Ideas modules.

Idea Object Schema

FieldTypeDescription
idstringUnique identifier (e.g. "idea-ai-onboarding")
titlestringShort, descriptive title
descriptionstringFull description of the idea
categorystringOne of: Product, Content, Business, Technical, Creative
statusstringOne of: new, backlog, in-progress, completed, archived
sourcestringOrigin: "generated", "community", "manual"
feasibilitynumber1–10 score for technical feasibility
impactnumber1–10 score for business/user impact
effortstring"Low", "Medium", or "High"
compositeScorenumberWeighted score (see Scoring Algorithm)
tagsstring[]Freeform tags for filtering
similarSolutionsstring[]Existing products/tools that solve similar problems
relatedIdeasstring[]IDs of related ideas
resourcesstring[]Reference URLs
notesstringAdditional context or implementation notes
Example JSON 
{
  "id": "idea-ai-onboarding",
  "title": "AI-Powered Client Onboarding Flow",
  "description": "Build an intelligent onboarding wizard...",
  "category": "Product",
  "status": "new",
  "source": "generated",
  "feasibility": 8,
  "impact": 9,
  "effort": "High",
  "compositeScore": 8.7,
  "tags": ["ai", "onboarding", "automation"],
  "similarSolutions": ["Typeform", "Intercom"],
  "relatedIdeas": ["idea-cms-templates"],
  "resources": ["https://openai.com/api/"],
  "notes": "Could integrate with Contentful setup."
}

⚖️ Scoring Algorithm

How the composite score is calculated for each idea.

compositeScore = (feasibility × 0.4) + (impact × 0.6) // Example: feasibility = 8, impact = 9 compositeScore = (8 × 0.4) + (9 × 0.6) = 3.2 + 5.4 = 8.6

Why This Weight?

Impact is weighted 50% higher than feasibility because the best ideas are the ones that move the needle — even if they're harder to build. Feasibility keeps the score grounded so pie-in-the-sky ideas don't dominate.

Filtering by Score

The Ideas Summary widget shows ideas with a composite score ≥ 9 or a 5-star personal rating. This ensures both objectively strong and personally important ideas surface on your dashboard.
